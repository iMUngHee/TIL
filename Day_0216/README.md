## 🏆 소감

가상 돔 기반의 웹 프레임워크는 익숙하여 도움이 많이 되었다. 하지만 러스트에 대한 이해가 부족하여 구현을 하는데 많은 시간이 걸렸다. 타입스크립트로 개발을 하는 것과 비슷한 느낌이 들면서도 웹이 아닌 C++로 시스템 프로그래밍을 하는 느낌이 강하게 들었다.
